AI with Rocky is a **modern AI starter** built with Next.js 15.

It integrates Next.js 15, Prisma 6, Neon, Auth.js v5, Resend, React Email, Shadcn/ui.

With **Authentication, User Roles & Dashboard**, you get a solid foundation for your next project.

This documentation is your go-to resource for configuring and using the starter effectively.

Let's get started and happy coding!
